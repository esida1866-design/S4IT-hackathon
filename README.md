# 🌬️ Bucharest Air Quality — SHIFT4IT Starter

This is the starter repo for the SHIFT4IT mini-hackathon. By the end you'll have a **live web app** showing real-time air quality across Bucharest — a URL you can text to anyone.

## How to use this repo

1. **Fork it** to your own GitHub account (button at top right of the GitHub page)
2. **Clone your fork in VS Code** — `Ctrl+Shift+P` → "Git: Clone"
3. **Open the deck** and follow the walkthrough step by step
4. **Fill in the 6 TODOs** inside `app.py` (search for `=== TODO`)
5. **Deploy** to Streamlit Cloud
6. **Send the link to your mom** 📱

## What's in here

```
app.py                 ← The main file you edit (6 TODOs)
data_fetcher.py        ← API + caching + fallback (already done)
requirements.txt       ← Python libraries Streamlit Cloud needs
sample_data.csv        ← Offline data so the app always works
stretch/               ← Bonus features (after core is done)
.gitignore             ← Files git should ignore
README.md              ← This file
```

## ⚠️ Important

**Never put your API key directly in `app.py`.** It goes in Streamlit Cloud's Secrets panel. The deck shows you exactly where.

## When you're stuck

1. Refresh the Streamlit Cloud page
2. Undo your last change (`Ctrl+Z`)
3. Ask the person next to you
4. Then raise your hand 🙋‍♀️

Now open the deck. Let's build.
